Ayuda y Documentación

Información sobre el uso del panel de administración

¿Cómo usar este panel?

Este panel te permite gestionar todas las configuraciones del sistema SICODIS sin necesidad de modificar código.

Persistencia de datos

Actualmente, todas las configuraciones se guardan en localStorage del navegador. Esto significa que:

  • Los cambios son inmediatos y locales a tu navegador
  • Si limpias el caché del navegador, se perderán las configuraciones personalizadas
  • Los cambios NO se sincronizan con otros usuarios
  • En el futuro, esto se migrará a una base de datos para sincronización global

Banner Inteligente

El banner utiliza un sistema de tracking que controla:

  • Frecuencia diaria: Cuántas veces mostrar el banner por día
  • Días consecutivos: Máximo de días seguidos que se muestra
  • Rango de fechas: Solo se muestra entre fecha_inicio y fecha_fin
  • Reset automático: Al cambiar la configuración, se resetea el tracking

Próximas funcionalidades

  • Edición inline de configuraciones
  • Historial de cambios y auditoría
  • Rollback a versiones anteriores
  • Sincronización con backend
  • Permisos de usuario (solo admin puede editar)
Nota importante: Este panel está en desarrollo. Actualmente solo muestra las configuraciones existentes. La funcionalidad de edición se agregará en los próximos sprints.
