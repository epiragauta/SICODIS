## Formato de Entradas Futuras

Usar las siguientes categorías:
- **Agregado**: Para nuevas funcionalidades o secciones
- **Modificado**: Para cambios en funcionalidad existente
- **Deprecado**: Para funcionalidad que será removida
- **Eliminado**: Para funcionalidad removida
- **Corregido**: Para correcciones de errores
- **Seguridad**: Para vulnerabilidades corregidas

### Ejemplo de Entrada:
```markdown
## [1.1.0] - YYYY-MM-DD
### Agregado
- Capítulo 08: Nuevo módulo de indicadores de gestión

### Modificado
- Capítulo 03-01: Actualización de screenshots con nueva interfaz
- Capítulo 06-01: Ampliación del glosario con 15 nuevos términos

### Corregido
- Capítulo 04-03: Corrección de fórmula de cálculo de AD 20%
```
